# 云道智能 · 设计优势展示

品牌设计岗作品展示页，可在 GitHub 上直接打开使用。

## 本地预览

用浏览器打开 `index.html` 即可。

## 在 GitHub 上在线查看

1. 将此文件夹整体上传到你的 GitHub 仓库（可新建空仓库后上传）。
2. 打开仓库 → **Settings** → **Pages**。
3. 在 **Source** 中选择 **Deploy from a branch**，分支选 `main`（或 `master`），目录选 **/ (root)**，保存。
4. 等待 1–2 分钟后，访问：`https://你的用户名.github.io/仓库名/` 即可看到页面。

也可用 [HTML Preview](https://htmlpreview.github.io/)：  
`https://htmlpreview.github.io/?https://github.com/你的用户名/仓库名/blob/main/index.html`
